Objective

Navigate the snake to eat as many apples (red blocks) as possible without accidentally running into yourself or the walls!

Instructions

Use the arrow keys to navigate the snake.

Eat red block to increase your score!

If you hit a wall, you WILL die.

If you "accidentally" eat yourself, you WILL die.

Press the start button when you're ready!

To start a new game hit Reset and Start again.
